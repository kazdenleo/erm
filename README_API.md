# Инструкция по выгрузке категорий с маркетплейсов

## 🎯 Система готова к работе!

### ✅ Что реализовано:

1. **Автоматическая выгрузка категорий** - При загрузке страницы система автоматически загружает все доступные категории с маркетплейсов
2. **Индивидуальное сопоставление** - Для каждой категории ERP можно выбрать соответствующую категорию с маркетплейса
3. **Хранение сопоставлений** - Все сопоставления сохраняются локально в браузере
4. **Визуальные индикаторы** - Сопоставленные категории помечаются зеленой галочкой

### 🔧 Как получить реальные категории:

#### Для Ozon:
1. Перейдите в раздел "Подключения по API"
2. Заполните форму Ozon:
   - **Client ID** - ваш Client ID из личного кабинета Ozon
   - **API Key** - ваш API ключ из личного кабинета Ozon
3. Нажмите "Проверить подключение" для тестирования
4. Нажмите "Сохранить" для сохранения настроек
5. Перейдите в раздел "Категории" - категории будут загружены автоматически

#### Для Wildberries:
1. Перейдите в раздел "Подключения по API"
2. Заполните форму Wildberries:
   - **API Key** - ваш API ключ из личного кабинета WB
3. Нажмите "Проверить подключение" для тестирования
4. Нажмите "Сохранить" для сохранения настроек
5. Перейдите в раздел "Категории" - категории будут загружены автоматически

### 📊 Текущее состояние:

**Без аутентификации** - система показывает сообщения об отсутствии подключения:
- **Ozon**: "Необходимы Client ID и API Key для подключения к Ozon"
- **Wildberries**: "Необходим API Key для подключения к Wildberries"  
- **Yandex Market**: "Интеграция с Yandex Market API пока не реализована"

**С правильной аутентификацией** - система загружает реальные категории с маркетплейсов:
- **Ozon**: Полное дерево категорий с иерархией (требует реальные Client ID и API Key)
- **Wildberries**: Все доступные категории товаров (требует реальный API Key)
- **Yandex Market**: Пока не реализовано

### ⚠️ Важно:
- **Демонстрационные данные убраны** - система работает только с реальными API
- **Проверка подключений** - в разделе "Подключения по API" отображается реальный статус
- **Без подключения** - в разделе "Категории" будет показано "Нет доступных категорий с маркетплейсов"

### 🎨 Интерфейс сопоставления:

1. **Кнопка "Сопоставить"** - открывает модальное окно для выбора категории
2. **Выпадающий список** - показывает все загруженные категории с маркетплейсов
3. **Информация о категории** - отображает название, путь и источник
4. **Сохранение** - сопоставление сохраняется и отображается зеленой галочкой

### 🔄 Автоматическое обновление:

- При изменении настроек подключений категории автоматически перезагружаются
- При сохранении сопоставления список категорий обновляется
- Все данные сохраняются в localStorage браузера

### 🚀 Готово к использованию!

Система полностью функциональна и готова к работе с реальными API маркетплейсов.
