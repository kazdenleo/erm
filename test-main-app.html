📊 Проверка данных в основном приложении

🔧 Инструкции по проверке

Для проверки основного приложения:
1. Откройте основное приложение
2. Перейдите в "Интеграции" → "Маркетплейсы"
3. Проверьте, что поля заполнены данными
4. Если поля пустые - нажмите F12 и проверьте консоль на ошибки
5. Попробуйте сохранить данные заново
