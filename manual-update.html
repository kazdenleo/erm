🔧 Ручное обновление данных на сервере

⚠️ Важно: Замените "ВАШ_РЕАЛЬНЫЙ_API_КЛЮЧ" на настоящие данные из вашего аккаунта Ozon.

📋 Шаг 1: Откройте консоль браузера

1. Откройте http://localhost:3001

2. Нажмите F12 или Ctrl+Shift+I

3. Перейдите на вкладку Console

🚀 Шаг 2: Обновите данные Ozon

Скопируйте и выполните этот код в консоли:

// Обновление данных Ozon fetch("http://localhost:3001/api/data/ozon", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ client_id: "2667088", api_key: "ВАШ_РЕАЛЬНЫЙ_API_КЛЮЧ_OZON" }) }).then(r => r.json()).then(result => { console.log("Результат обновления Ozon:", result); if (result.ok) { console.log("✅ Ozon данные обновлены успешно!"); } else { console.log("❌ Ошибка обновления Ozon:", result.message); } });

🧪 Шаг 3: Протестируйте товар AN1048

После обновления данных выполните этот код:

// Тестирование товара AN1048 fetch("http://localhost:3001/product/check/ozon?offer_id=AN1048") .then(r => r.json()) .then(result => { console.log("Результат проверки товара AN1048:", result); if (result.data.found) { console.log("🎉 ТОВАР AN1048 НАЙДЕН НА OZON!"); console.log("Детали товара:", result.data.product); } else { console.log("❌ Товар AN1048 не найден"); console.log("Причина:", result.data.message); } });

📂 Шаг 4: Проверьте категории Ozon

Загрузите категории с Ozon:

// Загрузка категорий Ozon fetch("http://localhost:3001/categories/ozon") .then(r => r.json()) .then(result => { console.log("Результат загрузки категорий:", result); if (result.ok && Array.isArray(result.data) && result.data.length > 0) { console.log(`🎉 ЗАГРУЖЕНО ${result.data.length} КАТЕГОРИЙ С OZON!`); console.log("Первые 5 категорий:", result.data.slice(0, 5)); } else { console.log("⚠️ Категории не загружены"); } });

📊 Шаг 5: Проверьте все данные на сервере

Посмотрите текущее состояние всех данных:

// Проверка всех данных на сервере fetch("http://localhost:3001/api/data") .then(r => r.json()) .then(result => { console.log("Все данные на сервере:", result); if (result.ok) { for (const [type, data] of Object.entries(result.data)) { if (Object.keys(data).length > 0) { console.log(`✅ ${type.toUpperCase()}: Настроено`); } else { console.log(`❌ ${type}: Не настроено`); } } } });
