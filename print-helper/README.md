# Print Helper — тихая печать этикеток после сборки

Локальный сервис для печати этикетки **сразу после сборки заказа**, без диалога в браузере.

**Запуск только через .exe.** На рабочих ПК Node.js не нужен — только папка с `erm-print-helper.exe` и `SumatraPDF.exe`.

---

## Как это работает

1. На каждом рабочем месте сборки запускается **erm-print-helper.exe** (на том же ПК, где открыт браузер с ERM).
2. Сервер ERM отдаёт клиентам URL помощника (`http://127.0.0.1:9100`) через **GET /api/config**.
3. Когда сборщик полностью собирает заказ, браузер отправляет запрос на localhost:9100 → Print Helper скачивает этикетку и печатает на **принтере по умолчанию** без диалога.

Если exe не запущен, открывается обычная страница печати в браузере (с диалогом).

---

## На каждом ПК сборки

1. Скопируйте папку с **erm-print-helper.exe** и **SumatraPDF.exe** (из `dist/` после сборки).
2. Запустите **erm-print-helper.exe** (окно можно свернуть).
3. В этой же папке должен лежать **SumatraPDF.exe**.

**Автозапуск:** ярлык на `erm-print-helper.exe` в папку «Автозагрузка» (`Win + R` → `shell:startup`).

**Требования:** Windows, принтер по умолчанию. Node.js не нужен.

---

## Сервер ERM (один раз)

В `.env` сервера добавьте:

```env
PRINT_HELPER_URL=http://127.0.0.1:9100
```

Перезапустите сервер. Клиенты получат этот URL через GET /api/config.

---

## Сборка .exe (для раздачи)

Сборка выполняется **один раз** на машине с Node.js (для разработки/IT):

```bash
cd testCursor/print-helper
npm install
npm run build:exe
```

В папке **`dist/`** появятся:

- **erm-print-helper.exe** — программа;
- **SumatraPDF.exe** — печать (копируется при сборке; если нет — скачайте с https://www.sumatrapdfreader.org/download-free-pdf-viewer и положите в ту же папку);
- **README.txt** — краткая инструкция для пользователя.

Раздайте папку **`dist/`** целиком. На рабочих ПК только её и запускают.

## Сборка установщика (Setup.exe)

Если нужен **один установочный файл**, используйте Inno Setup:

1) Установите **Inno Setup 6** на Windows-машине сборки (IT/разработчик).
2) Соберите exe и установщик:

```powershell
cd testCursor/print-helper
.\build-installer.ps1
```

Результат: `print-helper/installer/Output/erm-print-helper-setup.exe`

Чтобы раздать установщик через ERM, скопируйте его на сервер в:

`data/downloads/erm-print-helper-setup.exe`

## API

- **GET /print?orderId=...&labelUrl=...** — скачать этикетку по `labelUrl` и отправить на принтер по умолчанию.
- **GET /health** — проверка работы сервиса.
